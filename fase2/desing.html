Sintaxis y Selectores CSS

CSS es el formato recomendado para las paginas escritas en formato HTML en base a los estandares de "cascading style sheets" (hojas de estilo casada) publicado por el world wide web consortium (w3c). el uso de estandar web nos ahorra tiempo, da concistencia y facilita en gran medida la escritura de paginas web 1.Selector universal Sintaxis: * { atributo:valor; } Ejemplo:* { color: grey; }/* El estilo se aplicará a todos los elementos de la página*/ 2.Selector etiqueta Sintaxis: etiqueta { atributo:valor } Ejemplo: p {color: green;} /* El estilo se aplicará a todos los elementos

.*/ 3.Selector clase Sintaxis: .clase { atributo:valor } Ejemplo: .blend{color: red;} /* El estilo se aplicará a cualquier elemento que tenga la clase .blend */ 4.4. Selector identificador El selector identificador utiliza el atributo id para seleccionar un elemento. Solo puede haber un elemento con un id dado en un documento. Sintaxis: #id { atributo:valor } Ejemplo: #cent {color: blue;} /* El estilo se aplicará al elemento que tenga el id #cent */ 5. Selector descendiente Un elemento es descendiente de otro cuando se encuentra entre las etiquetas de apertura y de cierre del elemento padre. Sintaxis: selector1 selector2 selectorN {atributo: valor;} /* El estilo se aplica sobre el selector N */ Ejemplo: div p { color: black;} /* El estilo se aplica a todos los párrafos que se encuentren dentro de una etiqueta div */ 6. Combinación de selectores La combinación de selectores nos permite dar un estilo a todos los selectores indicados. Sintaxis: selector1, selector2, selector3{atributo: valor;} /* El estilo se aplica sobre los selectores indicados */ Ejemplo: div, p { color: orange;} /* El estilo se aplica a todos los divs y párrafos */ 7.Selector de hijos Se usa para seleccionar un elemento que es hijo de otro elemento. Sintaxis: selector1 > selector2 {atributo: valor;} /* El estilo se aplica sobre el selector 2 */ Ejemplo: div > p { color: white;} /* El estilo se aplica a todos los párrafos que sean hijos de un div */ 8.Selector adyacente Se usa para seleccionar elementos que son hermanos, es decir, su elemento padre es el mismo y están seguidos en el código HTML. Sintaxis: selector1 + selector2{ atributo: valor; } /* El estilo se aplica al selector 2 */ Ejemplo: div + p { color: black;} /* El estilo se aplica a todos los párrafos que sean hermanos de un div */ Durango, A. (2015). Diseño Web con CSS: 2ª Edición. IT Campus Academy.
